Project op

Project OP was mijn tweede project die ik heb gemaakt, dit project had ik gemaakt voor het Media College, specifiek voor de Media Developer opleiding. We moesten een landingspage maken waarop toekomstige studenten konden zien wat de opleding inhield.

Tijdens dit project had ik een paar foute gemaakt, waaronder dat ik voor een card niet wist hoe ik het aan de rechter kant positioneerde en dus een foto ervan had gemaakt en die aan de rechter kant gezet.s

Maar omdat dit mijn tweede project was, vond ik dat ik het goed afliep en dat ik er niet al te veel problemen mee had.
