project covid

Project Covid is een site die gemaakt is tijdens de corona periode, omdat in die tijd er mensen zijn die hulp nodig hebben, en met deze site helpen wij die mensen de corona tijd door te komen,

Dit project was een groepsproject die ik samen met 2 andere developers heb gemaakt, we hebben hierbij de taken goed onderveelt met user stories, zodat het duidelijk was wie wat moest doen en waarom we het moesten doen.

Tijdens dit project liepen wij vast met het PHP / API gedeelte, omdat we daar nieuw mee zijn en we het voor de eerste keer gebruikten.
