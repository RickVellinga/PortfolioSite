Wordpress thema

Wordpress thema heb ik gemaakt voor een schoolproject, dit is het eerste project wat ik had gemaakt op wordpress en was dus nieuw om een nieuw thema te leren maken.

Aan het begin liep ik vast, omdat ik niet echt begreep hoe ik wordpress moest gaan installeren met de databases, ook was er allemaal speciale code die wordpress gebruikt om bijvoorbeeld hun navigatie bar zichtbaar te maken.

Aangezien dit mijn eerste wordpress ervaring was, vond ik het best goed gaan omdat je eigenlijk gewoon een soort template / thema maakt die je vaker kan gebruiken in wordpress.
